O que é HTML?

(Voltar)

O HTML é uma linguagem de marcação de hipertexto usada para criação de páginas web. O HTML possui diversos elementos usados para delimitar ou agrupar diferentes partes do conteúdo de uma página web, e é interpretado por navegadores.

Tags

(Voltar)

As tags são elementos responsáveis pela formatação do texto dentro do HTML. O navegador será responsável por interpretar e produzir a estrutura do conteúdo visual da página.

A principal característica das tags é estarem sempre dentro dos sinais de chevron (sinal de “maior que” e “menor que”), ou seja: < >. As tags HTML são divididas em dois tipos: as que precisam de fechamento e as que não precisam de fechamento.

Texto

(Voltar)

No HTML podemos encontrar tags especificas para a formatação de texto. As tags de texto tem função de delimitar parágrafos, títulos, mudar cor, tamanho das letras e entre outras funções.

Links

(Voltar)

Os links tem a função de dentro da página web entrar em outras páginas, ou interagir com elementos da própria página.
